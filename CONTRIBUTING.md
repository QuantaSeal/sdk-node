# Contributing to the QuantaSeal Node.js SDK

## Development setup

```bash
git clone https://github.com/quantaseal/sdk-node.git
cd sdk-node
npm install

# Run tests
npm test

# Build
npm run build

# Lint
npm run lint
```

## Pull request guidelines

- Target the `main` branch
- Tests must pass (`npm test`)
- TypeScript strict mode - no `any` without justification
